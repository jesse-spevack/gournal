Scalable Design Pattern:
┌─ Habit Columns ─────────────────────────────────────────────────────┐
│  Each habit follows same pattern:                                   │
│  • 4px gap between habit columns                                    │
│  • First column includes day numbers (1-31)                         │
│  • Subsequent columns are checkboxes only                           │
│  • All use same header rotation (-90°) and alignment                │
└─────────────────────────────────────────────────────────────────────┘
┌─ Reflection Column ─────────────────────────────────────────────────┐
│  • Always separated by 8px gap from last habit                      │
│  • Uses flex-grow: 1 to fill remaining space                        │
│  • Horizontal header alignment                                      │
│  • Single-line text with ellipsis truncation                       │
└─────────────────────────────────────────────────────────────────────┘

Reflection Text Behavior

Single Line Truncation (12px Courier New)
This is a very long reflection that demonstrates how text gets truncated with an ellipsis when it exceeds the available width in the reflection column
  • white-space: nowrap - Prevents line wrapping
  • overflow: hidden - Hides overflow text
  • text-overflow: ellipsis - Adds "..." for truncated text
  • Perfect 1:1 alignment with checkbox rows (~26px height each)
